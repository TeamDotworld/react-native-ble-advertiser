# react-native-ble-advertiser

Advertise given message using BLE

## Installation

```sh
npm install react-native-ble-advertiser
```

## Usage

```js
import BleAdvertiser from 'react-native-ble-advertiser';

// ...

const result = await BleAdvertiser.setUserId('test');
```
